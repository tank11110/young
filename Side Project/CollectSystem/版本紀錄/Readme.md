## ver 0.1  
### 系統更新  
#### 一. 主畫面介面相關：  
	1. 調整書本資訊顯示內容為：書名、種類、集數  
#### 二. 功能列介面相關：  
	1. 調整功能列表為：首頁、搜尋紀錄、上傳藏書、登出  
	2. 整體寬度增加  
#### 三. upload介面相關：
	1. 調整為直式輸入介面  
	2. 新增查詢框    
	3. 新增輸入資訊：類型(單本、特裝版、限定版、套書)  
	4. 修改集數輸入資訊：可依據類別輸入不同類型的集數資訊  
		*說明：選擇套書類別將可在集數框輸入文字(ex. 1~10)  
#### 四. search介面相關：  
	1. 調整書本資料顯示方式  
	2. 新增查詢框  
	3. 調整書本顯示資訊：名稱、種類、集數、價格、購買時間
