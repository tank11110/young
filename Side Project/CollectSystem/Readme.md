# CollectSystem(收藏系統)

本系統用意為紀錄個人實體藏書。
提供搜尋功能避免購買重複的書籍、上傳功能紀錄購買過的書籍
希望可以提供比excel、手機記事本更好的紀錄方式
未來預計陸續提供各式各樣的蒐藏項目、簡易的圖表、數據分析
