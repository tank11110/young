五. 結論
-----------------------------------------------

(一) 可以使用模型預測新手機在市場上可被接受的價格

(二) 發現民眾最關心手機規格中的容量、電池大小、螢幕長寬，是影響價格的重量因素

(三) 藍芽、4G、觸控螢幕、wifi功能的重要程度為0。並非代表完全不重要
     這是現在手機的"必備"條件，因此民眾不會特別關心。
