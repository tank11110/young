題目：分析101營收與承租商家之關聯

但因無法拿到確切的營收資料和各年度商家資料，因此僅分析2020年的資料


研究動機：
-------------------------------------------------------------------------------------
	
     查詢跟101大樓相關的資料時發現查詢到的標題大多都是在疫情之下101營收高達150億…等。
     且文章內容有提到101奠定國際高端精品購物中心的定位，將持續與國際品牌合作，共同經營高端客源。
     我們對此感到興趣，想透過數據分析來研究101的高端精品購物中心的定位，以及營收和不同類別的商家組成是否有關係。

資料集說明：
-------------------------------------------------------------------------------------

	資料集的內容主要是從101的官方網站(https://www.taipei-101.com.tw/tw/)中的樓層導覽把所有出現的品牌爬取下來。
 	大類別是我們自行分類，主要分成精品/包包/珠寶和食物兩個類別。細項分類的方式則是套用101官網的分類。
  	國家欄位則是查詢該商家是哪個國家的品牌。
   	價位的欄位則是分成了三個類別。分別為：
    		低：商品價格中位數低於一千元以下。
    		中：商品單價中位數位在五千元到一千元內。
     		高：單價超過五千元以上的商品。
    在分析中我們目前只使用精品/包包/珠寶的大類。

 	此圖則是顯示出101大樓所有類別的價格分佈，我們發現101的商家大部分都是紅色(高價格店家)。藉此推斷101高端精品購物中心的定位。
  	同時中間價格的商品卻也不佔少數，我們認為101也想提供服務給數量更為龐大的中間價位的客戶，以此來增加收入。

   	若是透過分析發現光顧101的客戶中對於國家品牌有一定支持度，則可以參考此圖適當調整與該國家商家合作的數量。

	此圖則是希望將營收跟店家比例進行分析，觀察不同營收下的商家比例是否相同，以此來做為調整商家比例的依據。
 	透過價格等級來觀察說不同價位的店家數量是否對營收造成影響。可以適當的調整不同價位店家的比例。

  結論：2020年101的營收在疫情之下仍高達150億，表示其走向高級銷售百貨公司地位的策略是成功的
